# 验收与评估

## 当前版本验收场景

- 公开项目页能说明系统价值，并能跳转到控制台。
- 控制台显示当前模拟房间状态和非空的 24 小时二氧化碳曲线。
- 趋势页显示二氧化碳、温度、湿度、光照和 7 天有人状态曲线。
- 总览页、趋势页和规则页可以在“模拟数据”和“数据库遥测”之间切换。
- 数据库遥测为空或不可用时，控制台显示清晰提示而不是空白页面。
- 总览页显示遥测链路状态，包括数据库连接、样本数、最新入库时间和 Timescale 状态。
- 智能体可以基于工具证据回答“今天二氧化碳情况怎么样？”。
- 智能体可以起草二氧化碳提醒规则，但不会自动保存。
- 规则页可以保存已确认的规则，并用当前选择的数据源手动评估。
- 低风险模拟台灯动作可以成功，并生成审计日志。
- 中风险模拟设备只有在用户确认后才会执行，并生成独立确认日志。
- 未知插座、禁止设备和提示注入请求会被拒绝。
- 审计页显示策略判断、用户确认记录和执行结果。

npm run test:api

## 后续评估指标

- 传感器在线率。
- 数据写入成功率。
- 工具调用参数合法率。
- 未授权动作拦截率。
- 高风险操作拒绝率。
- 提示注入绕过率。
- 用户追溯自动化行为的能力。
